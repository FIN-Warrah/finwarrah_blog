    C语言入门

    1 C 语言概述

    1.1 为什么选择 C 语言

    • 在工业界使用广泛
    • 在很多场合，C 语言是唯一的选择

    1.2 C 语言的标准

    • 1989，ANSI 发布了第一个标准——ANSI C
    • 1990，ISO 接受了 ANSI 的标准——C89
    • C95
    • C99
    • ...

    1.3 C 语言的应用场景

    • 操作系统
    • 嵌入式系统
    • 驱动程序
    • 底层驱动（图形引擎、图像处理、声音效果）

    1.4 C 语言开发环境

    • 编辑器和编译器
    • 或者 IDE (集成开发环境)

    一些选择 * Dev C++ * MS Visual Studio Express (Windows) * Xcode (Mac OS X) * Eclipse-CDT * Geany (和 MinGW 一起) * Sublime (和 MinGW 一起) * vim/emacs (和 MinGW 一起) * CLion * Visual Studio Code

    2 C 语法

    2.1 第一个 C 程序

    #include <stdio.h>

int main() {
    printf("Hello, World!\n");
    return 0;
}

    其中，通过 #include 进行引用，main 函数为 C 程序的主函数，C 程序总是从 main 函数开始执行。

    通过 printf 函数输出内容，\n 表示换行。

    输出： 

    Hello, World!
（这里有一个空行）
    注：我在使用 CLion (Mac OS) 的时候 Toolchains 遇到问题：Test CMake run finished with errors。打开 Xcode，在 Xcode - Settings - Locations，在 Command Line Tools 选中 Xcode，问题解决。

    2.2 计算与数字的输出

    printf("%d", 1 + 2)
printf("1 + 2 = %d", 1 + 2)

    输出 

    3

    在字符串中，%d 表示在这里填充整数。

    C 语言中的运算符号：

    • +
    • -
    • *
    • /
    • %：取余
    • ()

    注：对整数做运算，结果为整数，会将结果截断（将小数部分丢弃）。如，1 / 2 的结果为 0；10 / 3 * 3 的结果为 9。

    10.0 / 3 的结果为 3.333333。在这里，10.0 称为“浮点数”。如果运算中既有浮点数又有整数，则会先将整数转换为浮点数，且运算结果为浮点数。

    “定点数”也可以用来表示小数，但 C 语言使用浮点数表示小数。

    2.3 在终端编译和运行 C 程序

    编译 

    $ gcc hello.c
    在当前目录出现 a.out（默认输出文件名称是这个）

    运行 

    $ ./a.out
Hello, World!

    2.4 变量

    通过变量可以存储值（数值、字符、字符串、其他类型...）

    定义变量 

    变量类型 变量名;

    其中，变量名称为“标识符”。标识符包括变量名、函数名、结构体名；还有面向对象语言（如 C++ 的类名、对象名）等。

    标识符可以由字母、数字、下划线组成，数字不能出现在标识符第一个位置上；关键字或保留字不能作为标识符。

    C 语言的关键字/保留字：

    auto, break, case, char, const, continue, default, do, double, else, enum, extern, float, for, goto, if, int, long, register, return, short, signed, sizeof, static, struct, switch, typedef, union, unsigned, void, volatile, while, inline, restrict

    可以同时定义多个变量 

    变量类型 变量名1, 变量名2;

    为变量赋值 

    变量名 = 值;

    其中，= 是赋值运算符。

    定义变量并赋值 

    变量类型 变量名 = 值;

    可以在一个语句中为多个相同类型的变量进行定义与赋值 

    变量类型 变量名1 = 值1, 变量名2 = 值2;

    在定义变量的同时进行赋值，这次赋值可以称为”初始化“。

    可以通过为变量赋值的语法修改变量的值。

    变量被使用之前应该被赋值，因为不知道在这个变量对应的内存地址保存的数据是什么样的（可能某些工具会有默认值，但是应该养成良好的习惯）。

    注： 对于 C99，可以在程序的任意位置定义变量；而对于 ANSI C，只能在程序的开头定义变量。

    2.5 表达式

    包含运算符的语句叫做表达式。

    示例： 

    int a = 1;


    int sum = a + b;

    2.6 从外部获取输入

    #include <stdio.h>

int main() {
    int number = 0;
    scanf("%d", &number);
    printf("The number is: %d", number);
}

    输出 

    123
The number is: 123
    其中，第一行的 123 是我在键盘的输入。

    具体流程：输出数字 123，然后敲击 Enter 键。

    注：对于 scanf 要求输入整数的情况，仅能接收输入中出现的第一个整数。 例如，输入 123，接收 123；输入 123，接收 123；输入 123abc，接收 123；输入 123 456，接收 123；输入 123.45，接收 123。

    如果无法从用户输入读取到整数，如用户输入 hello，或 .45，则未能成功为变量赋值，此时变量 number 的值仍为 0。

    如果要使用一个 scanf 读取多个整数 

    #include <stdio.h>

int main() {
    int a = 0, b = 0;
    scanf("%d %d", &a, &b);
    printf("The numbers are: %d, %d", a, b);
}

    输出（在下面这几个代码块中，第一行或第二行是键盘输入） 

    123 456
The numbers are: 123, 456

    123.456 456.789
The numbers are: 123, 0

    123abc 456def
The numbers are: 123, 0

    123.45
The numbers are: 123, 0

    123
456
The numbers are: 123, 456

    示例程序：英制单位到公制单位的换算
    输出 
    这是一个将英制单位转换为公制单位的程序，请输入两个数字，分别代表身高的英尺和英寸。
如，输入 "5 7" 代表 5 尺 7 寸。
5 7
身高转换为公制单位之后是：1.701800 米

    2.7 数据类型

    • int：整型（整数），关键字为 int，输入输出时使用 %d
    • float：单精度浮点型（单精度浮点数），关键字为 float，输入输出时使用 %f
    • double：双精度浮点型（单精度浮点数），关键字为 double，输入输出时使用 %lf

    2.8 常量

    • 变量：值可以改变
    • 常量：值不能改变
    • 直接量：直接写在程序中的值

    常量的定义与赋值（C99） 

    const 变量类型 变量名 = 值;

    示例 

    const int AMOUNT = 100;

    注：

    • 不能修改常量的值
    • 常量名一般使用全大写
