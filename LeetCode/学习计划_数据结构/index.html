    217 存在重复元素

    解法一：使用 HashSet，遍历数组中全部元素，如果存在于 HashSet 中，则返回 true，否则添加到 HashSet 中。如果程序运行到遍历结束仍为返回，说明不存在出现两次的元素，返回 false。

    时间复杂度：O(N)

    import java.util.HashSet;

public class Solution {

    public boolean containsDuplicate(int[] nums) {
        HashSet<Integer> set = new HashSet<>();
        for (int i = 0; i < nums.length; i++) {
            if (set.contains(nums[i]))
                return true;
            else
                set.add(nums[i]);
        }
        return false;
    }
}

    简化写法 

    import java.util.HashSet;

public class Solution {

    public boolean containsDuplicate(int[] nums) {
        HashSet<Integer> set = new HashSet<>();
        for (int i = 0; i < nums.length; i++) {
            if (!set.add(nums[i]))  // Returns true if this set did not already contain the specified element
                return true;
        }
        return false;
    }
}

    53 最大子数组和

    解法一：一次遍历。用两个变量 seqSum 和 res 分别记录当前子数组的和以及到目前的最大结果。遍历到每个元素时，对当前子数组元素之和进行检查，如果当前元素值大于当前子数组的和，说明该子数组到当前值之前的部分对子数组的总和起反作用，因此对子数组进行重置，令子数组仅包含当前元素。然后检查当前子数组元素之和是否大于到目前的最大结果，如果大于，则令最大结果为当前子数组元素之和。

    时间复杂度：O(N) 

    class Solution {
    public int maxSubArray(int[] nums) {
        int seqSum = nums[0];
        int res = nums[0];
        for (int i = 1; i < nums.length; i++) {
            seqSum += nums[i];
            if (nums[i] > seqSum)
                seqSum = nums[i];
            if (seqSum > res)
                res = seqSum;
        }
        return res;
    }
}

    简化写法 

    class Solution {
    public int maxSubArray(int[] nums) {
        int seqSum = nums[0], res = nums[0];
        for (int i = 1; i < nums.length; i++) {
            seqSum = Math.max(seqSum + nums[i], nums[i]);
            res = Math.max(res, seqSum);
        }
        return res;
    }
}

    1 两数之和

    解法一：哈希表，遍历数字中全部元素，每次将当前元素作为键、将当前索引作为值保存到哈希表中。如果遍历到某一个元素时发现当前元素与哈希表中某个键相加为 target，说明找到答案，返回结果。

    时间复杂度：O(N)

    class Solution {
    public int[] twoSum(int[] nums, int target) {
        Map<Integer, Integer> map = new HashMap<>();
        for (int i = 0; i < nums.length; ++i) {
            if (map.containsKey(target - nums[i])) {
                return new int[]{map.get(target - nums[i]), i};
            }
            map.put(nums[i], i);
        }
        return null;
    }
}

    88 合并两个有序数组

    解法一：逆序遍历。分别使用一个索引指示两个数组的最后一个元素，每次将其中较大的元素更新到 nums1。如果遍历完 nums1 后  nums2 没有遍历完，则将  nums2 中剩下的元素更新到  nums1 中。

    时间复杂度：O(m+n)

    class Solution {
    public void merge(int[] nums1, int m, int[] nums2, int n) {
        int position1 = m - 1, position2 = n - 1;
        int positionRes = nums1.length - 1;
        boolean arr1;
        while (position1 >=0 && position2 >= 0) {
            arr1 = nums1[position1] >= nums2[position2];
            if (arr1) {
                nums1[positionRes] = nums1[position1];
                position1 -= 1;
            } else {
                nums1[positionRes] = nums2[position2];
                position2 -= 1;
            }
            positionRes -= 1;
        }
        for (int i = 0; i <= position2; i++) {
            nums1[i] = nums2[i];
        }
    }
}

    350 两个数组的交集 II

    解法一：一次遍历。遍历 nums1，使用哈希表存放  nums1 中元素值及出现的次数。遍历  nums2，如果遍历到的数值在哈希表中存在，则将当前数值添加到输出数组中，然后哈希表中当前数值出现次数减一，如果此时当前数值出现次数为 0，则将其从哈希表中删除。

    时间复杂度：O(m+n)

    class Solution {
    public int[] intersect(int[] nums1, int[] nums2) {
        HashMap<Integer, Integer> map = new HashMap<>();
        ArrayList<Integer> res = new ArrayList<>();
        for (int i = 0; i < nums1.length; i++) {
            if (!map.containsKey(nums1[i]))
                map.put(nums1[i], 1);
            else
                map.put(nums1[i], map.get(nums1[i]) + 1);
        }
        for (int i = 0; i < nums2.length; i++) {
            if (map.containsKey(nums2[i])) {
                    res.add(nums2[i]);
                    map.put(nums2[i], map.get(nums2[i]) - 1);
                    if (map.get(nums2[i]) == 0)
                        map.remove(nums2[i]);
            }
        }

        int[] resArray = new int[res.size()];
        for (int i = 0; i < res.size(); i++) {
            resArray[i] = res.get(i);
        }
        return resArray;
    }
}

    121 买卖股票的最佳时机

    解法一：用两个变量分别保存最小值对应的索引和最大值对应的索引（要求最小值对应的索引小于等于最大值对应的索引），用一个变量存储结果。遍历数组中的每个元素，如果当前元素小于之前的最小值，则更新最小值索引到当前位置，同时更新之前得到的最大结果，同时更新最小值对应的索引和最大值对应的索引。如果当前值大于之前的最大值，则更新最大值。

    时间复杂度：O(N)

    class Solution {
    public int maxProfit(int[] prices) {
        int indexMin=0, indexMax=0;
        int res = 0;
        for (int i = 1; i < prices.length; i++) {
            if (prices[i] < prices[indexMin]) {
                res = Math.max(res, prices[indexMax] - prices[indexMin]);
                indexMin = i;
                indexMax = i;
            } else if (prices[i] > prices[indexMax]) {
                indexMax = i;
            }
        }
        res = Math.max(res, prices[indexMax] - prices[indexMin]);
        return res;
    }
}

    解法二：保存最小值，遍历数组，更新最小值，并计算如果当前卖出的收益。

    时间复杂度：O(N)

    在 Leetcode 提交，使用时间比解法一更短。

    class Solution {
    public int maxProfit(int[] prices) {
        int minValue = Integer.MAX_VALUE;
        int res = 0;
        for (int i = 0; i < prices.length; i++) {
            if (prices[i] < minValue) {
                minValue = prices[i];
            } else if (prices[i] - minValue > res) {
                res = prices[i] - minValue;
            }
        }
        return res;
    }
}
